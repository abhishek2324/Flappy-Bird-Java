## Description
Flappy Bird is a classic arcade-style game where the player controls a bird and tries to navigate through gaps between pipes.  
This project is implemented using **Java** and the **Swing framework**, focusing on game logic, event handling, and basic graphics rendering.

---

## Features
- **Basic Gameplay**: Control the bird to fly between pipes.
- **Scoring System**: Earn points for successfully passing through pipes.
- **Pause / Resume**: Pause the game using the spacebar and resume from the same position.
- **Game Over Screen**: The game ends when the bird hits a pipe or the ground.
- **Restart Option**: Restart the game by pressing Enter after game over.
